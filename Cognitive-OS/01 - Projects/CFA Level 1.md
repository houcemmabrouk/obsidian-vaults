## 🧠 Routine journalière (ancrée neuroscience)

### Préparation mentale
- **10'** Push mail — zones d'incompréhension → *externalisation cognitive* ([[Cognitive Load Theory]])
- **5'** Lecture planning + objectif du jour → *planification métacognitive*

### Session matin · 09:00 → 12:00
| T | Action | Principe |
| --- | --- | --- |
| 30' | Encodage LM : Summary Notes FR + Audio | [[Dual Coding]] |
| 20' | Lecture Readings ENG + Blank Recall | [[Active Recall]] + contextualisation |
| 20' | Concept Mapping + Cold Test | [[Testing Effect]] |
| 30' | Synthesis LOS Sheet + Audio + Retest | Élaboration + test-retest |
| 20' | Pause cognitive | Consolidation hippocampique |
| 30' | Knowledge Audit + Weakness Pool | Métacognition + error-based learning |
| 30' | Targeted Revision + TDS Sheet | Remédiation ciblée |

### Session après-midi · 12:00 → 15:30
| T | Action | Principe |
| --- | --- | --- |
| 20' | Concept-on-Concept + Calculator Training | [[Interleaving]] + mémoire procédurale |
| 30' | Mock 20 questions + évaluation | Simulation + feedback immédiat |
| 20' | Pause cognitive | Récupération |
| 20' | Score Analysis + Essential Summary | Calibration métacognitive |
| 20' | Dashboard + Log of Result | Tracking + journal réflexif |
