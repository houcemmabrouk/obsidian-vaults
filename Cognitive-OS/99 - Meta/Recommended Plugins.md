# 🔌 Recommended Plugins

Plugins communautaires qui font tourner le Cognitive OS. Installer via `Paramètres → Modules complémentaires → Parcourir`.

## Essentiels

| Plugin | Rôle | Utilisé par |
| --- | --- | --- |
| **Dataview** | Requêtes dynamiques sur les notes (tables, listes, tâches) | [[🏠 Home]], [[📊 Dashboard]], [[✅ Tasks Dashboard]] |
| **Templater** | Modèles avancés (dates, prompts, scripts) | tout `08 - Templates` |
| **Calendar** | Vue calendrier + navigation daily notes | Journal |
| **Periodic Notes** | Daily / Weekly / Monthly automatisés | Journal |

## Confort

| Plugin | Rôle |
| --- | --- |
| **Tag Wrangler** | Renommer/fusionner les tags proprement |
| **Homepage** | Ouvrir [[🏠 Home]] au démarrage |
| **Style Settings** | Régler les thèmes CSS |
| **Excalidraw** | Schémas & croquis pour les MOCs |
| **Advanced Tables** | Édition confortable des tableaux Markdown |

## Révision (Zettelkasten actif)

| Plugin | Rôle |
| --- | --- |
| **Spaced Repetition** | Transformer les notes en flashcards SRS (voir [[Spaced Repetition]]) |

## Réglages core recommandés

- **Modèles (core)** : dossier `08 - Templates`.
- **Notes quotidiennes** : dossier `06 - Journal/Daily`, format `YYYY-MM-DD`, modèle `08 - Templates/Daily Note`.
- **Nouveaux liens** : *relatifs au fichier* désactivé → *chemin le plus court*.
- **Fichiers joints** : dossier `Attachments`.

> [!note]
> Les dashboards de ce vault utilisent **Dataview**. Sans lui, les blocs ```` ```dataview ```` s'afficheront comme du texte brut — c'est normal, installe le plugin.
