# Second Brain

Un **second brain** est un système externe et fiable de gestion des connaissances : il **stocke** ce que je ne peux pas retenir, pour libérer mon cerveau afin de **penser** et **créer**.

## Le fondement
La mémoire de travail est saturable ([[Cognitive Load Theory]]). Externaliser l'information = décharger cette mémoire → plus de ressources pour le raisonnement. *« L'esprit sert à avoir des idées, pas à les stocker. »*

## La méthode CODE (Tiago Forte)
1. **Capture** — garder ce qui résonne (Inbox).
2. **Organize** — ranger par actionnabilité ([[PARA Method]]).
3. **Distill** — condenser jusqu'à l'essentiel (notes progressives).
4. **Express** — produire : c'est le but final.

## Ce vault EST mon second brain
- **Capture** → `00 - Inbox` / Fleeting
- **Organize** → PARA + [[Zettelkasten Method]]
- **Distill** → notes permanentes + MOCs
- **Express** → projets, écrits, décisions
