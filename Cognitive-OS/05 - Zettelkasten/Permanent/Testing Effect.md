# Testing Effect

Le **testing effect** (ou *retrieval practice effect*) : passer un test sur une information **améliore** sa rétention future, davantage que de réétudier la même information pendant le même temps.

## L'idée en une phrase
Le test n'est pas qu'une mesure — c'est un **acte d'apprentissage** (voir [[Active Recall]]).

## Preuve classique
Roediger & Karpicke (2006) : les étudiants qui *se testaient* retenaient nettement plus à une semaine que ceux qui *relisaient*, alors même qu'ils se sentaient *moins* confiants. → La confiance est un mauvais indicateur d'apprentissage.

## Implications
- Privilégier **test > relecture**, toujours.
- Le **feedback** après le test amplifie l'effet.
- Fonctionne en synergie avec [[Spaced Repetition]] et [[Interleaving]].
