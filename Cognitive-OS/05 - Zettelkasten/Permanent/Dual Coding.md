# Dual Coding

**Dual coding** (Paivio) : combiner information **verbale** et **visuelle** crée deux voies de récupération distinctes → meilleur encodage et rappel.

## L'idée en une phrase
Un mot *plus* une image valent mieux que deux fois le mot.

## Pourquoi ça marche
- Deux traces mémorielles indépendantes (verbale + imagée) → plus de chemins d'accès.
- Allège la [[Cognitive Load Theory|charge cognitive]] quand le visuel *remplace* du texte redondant (attention : image + texte redondant = surcharge).

## En pratique
- Schémas, diagrammes, cartes conceptuelles (Excalidraw dans les [[07 - MOCs/🗺️ Master Index|MOCs]]).
- Coupler texte + **audio** (révision passive) — principe repris par [[Wingman v2]] et [[CognisAI (Learning OS)]].
- Concept mapping en révision (cf. routine [[CFA Level 1]]).
