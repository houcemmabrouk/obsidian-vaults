# Zettelkasten Method

Le **Zettelkasten** (« boîte à fiches », Niklas Luhmann) est un réseau de **notes atomiques** reliées, conçu pour *penser* et *produire*, pas seulement archiver.

## Trois types de notes
1. **Fleeting** — captures brutes, jetables après tri.
2. **Literature** — notes de lecture : idée d'une source, **avec mes mots** + référence.
3. **Permanent** — idées atomiques durables, autonomes, densément liées.

## Les 3 principes
- **Atomicité** — une note = une seule idée (réutilisable, liable).
- **Autonomie** — la note se comprend seule, hors de son contexte d'origine.
- **Connexion** — chaque note pointe vers d'autres ; la valeur naît du **réseau**, pas des notes isolées.

## Pourquoi ça produit des idées
Relier de force une nouvelle note aux anciennes crée des **collisions d'idées** inattendues. Le Zettelkasten devient un partenaire de pensée. Écrire avec ses mots = [[Active Recall]].

## Zettelkasten vs PARA
- [[PARA Method|PARA]] = **agir** (orienté projet).
- Zettelkasten = **savoir** (orienté idée).
- Les deux se rejoignent via les [[07 - MOCs/🗺️ Master Index|MOCs]].
