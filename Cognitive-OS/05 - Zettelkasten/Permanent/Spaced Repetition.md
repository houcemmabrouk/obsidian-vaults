# Spaced Repetition

**Spaced repetition** = espacer les révisions dans le temps, en allongeant l'intervalle à chaque succès. On révise **juste avant d'oublier**.

## L'idée en une phrase
Exploiter la **courbe de l'oubli** (Ebbinghaus) : réactiver une trace au bon moment la rend plus durable qu'une révision massée.

## Le mécanisme
- **Spacing effect** : des sessions distribuées > une session bloquée (bachotage).
- À chaque rappel réussi, l'intervalle suivant s'allonge (ex. 1j → 3j → 7j → 30j → …).
- Un échec raccourcit l'intervalle (retour au début).

## Algorithmes
- **SM-2** (SuperMemo) — base d'Anki et du plugin *Spaced Repetition* d'Obsidian.
- Variante **SM-2 étendu** utilisée dans [[CognisAI (Learning OS)]] pour piloter les flashcards par LOS.

## En pratique
- Coupler **toujours** avec [[Active Recall]] : espacer des *tests*, pas des relectures.
- Multi-intervalle typique : **J+1 / J+3 / J+7 / J+30**.
- Dans ce vault : plugin *Spaced Repetition* sur les notes permanentes que je veux mémoriser.
