# 🧠 Cognitive OS

> Mon système d'exploitation cognitif — un *second brain* pour capturer, organiser, connecter et retrouver tout ce qui compte.

Ce vault combine deux méthodes éprouvées :

- **PARA** (Projects · Areas · Resources · Archive) — pour l'**organisation orientée action**.
- **Zettelkasten** — pour la **pensée en réseau** et les connaissances durables.

Le tout piloté par un **journal** (daily/weekly/monthly), des **MOCs** (cartes de contenu) et des **dashboards** Dataview.

## 🗂️ Structure

```
Cognitive-OS/
├── 00 - Inbox/          # 📥 Capture rapide (à trier)
├── 01 - Projects/       # 🎯 Objectifs avec deadline (effort actif)
├── 02 - Areas/          # 🔁 Responsabilités continues (sans fin)
├── 03 - Resources/      # 📚 Thèmes & références utiles un jour
├── 04 - Archive/        # 📦 Inactif / terminé
├── 05 - Zettelkasten/   # 💡 Notes atomiques (Fleeting / Literature / Permanent)
├── 06 - Journal/        # 📓 Daily · Weekly · Monthly
├── 07 - MOCs/           # 🗺️ Maps of Content (index thématiques)
├── 08 - Templates/      # 🧩 Modèles (Templater / core templates)
├── 09 - Dashboards/     # 📊 Vues dynamiques (Dataview)
└── 99 - Meta/           # ⚙️ À propos, conventions, plugins
```
