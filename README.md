# 🧠 obsidian-vaults

Dépôt de sauvegarde de mes vaults Obsidian.

## Vaults

| Vault | Description |
| --- | --- |
| **[Cognitive-OS](./Cognitive-OS/)** | Mon *second brain* — système de gestion des connaissances (PARA + Zettelkasten), journal, projets et science de l'apprentissage. Voir [`Cognitive-OS/README.md`](./Cognitive-OS/README.md). |

## Sauvegarde

Le script [`backupvaults.bat`](./backupvaults.bat) commit et push automatiquement l'ensemble des vaults vers GitHub (double-clic depuis Windows).

## Ouvrir un vault

Dans Obsidian : `Ouvrir un autre coffre` → `Ouvrir un dossier existant comme coffre` → sélectionner le dossier du vault (ex. `Cognitive-OS/`).
